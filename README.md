An interactive dashboard that reveals the story behind retail profits, showing where sales are booming, where margins are thinning, and which customer segments drive the biggest gains.
With clear visualizations and data-driven insights, retail managers can spot opportunities, tackle problem areas, and shape smarter strategies to maximize growth and profitability.
Due to organizational restrictions on Power BI Service, public web embedding is currently disabled for this report. As a result, the dashboard cannot be viewed directly in the browser.
To explore the interactive dashboard:
1. Download both files from this repository.
2. Open the .pbix file using Power BI Desktop.
3. Navigate through the report pages to interact with filters, slicers, and visualizations.
Power BI Desktop is free to download from Microsoft's official website. This method ensures full access to all report features, visuals, DAX calculations, and interactive components exactly as designed.
